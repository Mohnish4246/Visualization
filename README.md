# Data Visualization Libraries

This folder contains notebooks related to Python data visualization libraries along with a sample dataset used for practice.

## 📂 Files

### 📘 `matplt.ipynb`
This notebook covers **Matplotlib**, one of the most popular Python libraries for data visualization.

**Matplotlib provides:**
- Basic data visualization tools
- Line charts
- Bar charts
- Pie charts
- Histograms
- Scatter plots
- Customization of graphs
- Labels, titles, legends, and grids
- Figure and subplot creation

---

### 📘 `sea.ipynb`
This notebook covers **Seaborn**, a high-level visualization library built on top of Matplotlib.

**Seaborn provides:**
- Attractive statistical visualizations
- Distribution plots
- Categorical plots
- Relationship plots
- Regression plots
- Heatmaps
- Pair plots
- Correlation visualization
- Better themes and styling
- Easy handling of Pandas DataFrames

---

### 📄 `students_markssss.csv`
This is a sample dataset used for practicing data visualization.

**The dataset provides:**
- Student-related information
- Numerical data for analysis
- Data for creating different charts and graphs
- Practice for comparing, analyzing, and visualizing data using Matplotlib and Seaborn
  
---

## 📌 Purpose
These notebooks help in learning how to visualize data effectively, understand patterns, compare values, identify trends, and present insights using Python visualization libraries.
